# 🚀 Modern AI Portfolio & Personal Brand

A professional, high-performance portfolio website built for developers. This project transforms a static site into a dynamic, data-driven application with a "GitHub-operated backend," automated deployments, and premium UI/UX.

![Portfolio Preview](https://via.placeholder.com/1200x600?text=Premium+Portfolio+Preview)

## ✨ Key Features

- **📂 GitHub-Based Backend**: Manage your projects, skills, and blog posts directly by editing `database.json`. No database setup required.
- **💬 Live Testimonials**: Built-in testimonial section with a professional submission modal and star ratings.
- **📬 Smart Contact Form**: Powered by Web3Forms (100% Free) with AJAX submission, auto-replies, and professional validation.
- **🤖 AI-Ready Automation**: 
  - **GitHub Actions Deployment**: Automatically deploys to GitHub Pages on every push.
  - **HTML Validation**: Automated CI/CD check to ensure W3C standard code quality.
- **🌗 Premium Dark Mode**: Smooth transition between light and dark themes with system preference detection.
- **🎨 Modern UI/UX**: Built with Tailwind CSS, Framer-inspired animations (AOS), and professional typography.

## 🛠️ Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript, Tailwind CSS (via CDN)
- **Animations**: AOS.js (Animate On Scroll), Typed.js
- **Backend/CMS**: `database.json` (Flat-file CMS)
- **Forms**: Web3Forms (Serverless)
- **CI/CD**: GitHub Actions

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/mukund-ahire/Personal-Website-Mukund-Ahire.git
cd Personal-Website-Mukund-Ahire
```

### 2. Update your data
Simply edit `database.json` to add your own projects, skills, and testimonials. The website will update automatically!

### 3. Activate the Contact Form
1. Get a free access key at [Web3Forms](https://web3forms.com).
2. Replace the `access_key` value in `index.html` (~line 434).

### 4. Deploy
Push your changes to the `main` branch. GitHub Actions will handle the deployment to GitHub Pages.
